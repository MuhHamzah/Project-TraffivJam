🚀 CARA MEMULAI:

WINDOWS:
1. Double-click: setup.bat
   (tunggu sampai selesai ~ 5-10 menit)

2. Edit file: templates/maps.html
   Ganti: key=YOUR_GOOGLE_MAPS_API_KEY
   Dengan API key dari Google Cloud

3. Double-click: run.bat

4. Buka browser: http://localhost:5000
   ✓ SELESAI!

LINUX/MAC:
1. chmod +x setup.sh run.sh

2. ./setup.sh
   (tunggu sampai selesai)

3. Edit file: templates/maps.html
   Ganti API key

4. ./run.sh

5. Buka browser: http://localhost:5000

⚙️ KONFIGURASI PENTING:

1. Google Maps API Key (Required untuk halaman Maps):
   - Daftar di: https://console.cloud.google.com/
   - Enable: Maps JavaScript API + Directions API
   - Buat API Key
   - Edit: templates/maps.html
   - Update: key=YOUR_KEY_HERE

2. Speed Calibration (Optional):
   - Edit: detector.py
   - Cari: CAMERA_CONFIGS
   - Adjust: pixels_per_meter (default: 50)
   - Measure jarak nyata di video untuk akurasi

3. Detection Sensitivity:
   - Edit: detector.py
   - Ubah: TRACKING_MAX_DISTANCE (default: 60)
   - TRACKING_TIMEOUT (default: 5 detik)

📊 DATA STORAGE:

Per-Minute CSV (setiap menit):
  Columns: timestamp, camera, cars, motorbikes, buses, trucks, avg_speed_kmh
  Location: data/{camera_id}/per_minute.csv
  Contoh:
    2026-02-04 10:30:00,jembatan_merah,5,12,2,1,35.5

Daily CSV (agregasi per hari):
  Columns: date, camera, cars, motorbikes, buses, trucks, total, avg_speed_kmh
  Location: data/{camera_id}/daily.csv
  Contoh:
    2026-02-04,jembatan_merah,120,280,25,8,433,38.2

🔍 MONITORING & TROUBLESHOOTING:

Check Health:
  http://localhost:5000/api/health
  Response: {status: ok, uptime_seconds: ..., workers: {...}}

Logs (CCTV workers):
  Ketika run.bat/run.sh, akan ada output:
  [jembatan_merah] Detections: 5 cars, 3 bikes...
  [jembatan_merah] Writing per-minute data...

Common Issues & Solutions:
  ❌ Video not playing
     → Check HLS stream URL accessible
     → Check internet connection
     → Check browser console (F12)

  ❌ Maps blank
     → Add Google Maps API key
     → Check API key is valid
     → Check APIs are enabled

  ❌ No detections
     → Wait 1-2 min for model to load
     → Check camera has vehicles
     → Check camera quality

  ❌ CPU high usage
     → Reduce detection frequency
     → Use GPU (install torch[cuda])

📈 PERFORMA:

  - RAM: ~200-400 MB (dapat bertambah dengan model)
  - CPU: 20-40% (per camera detection)

Detection Speed:

  - Detection interval: ~1 sec per frame
  - Processing lag: minimal

Scalability:
  - 3 cameras default (dapat ditambah)
  - Multi-threaded detection workers
  - Flask + Gunicorn untuk production
